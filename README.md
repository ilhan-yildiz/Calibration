# Kalibrasyon Takip Botu 🤖

Yıllık Kalibrasyon Takvimi'nde arama yapabilen Telegram botu.

## Özellikler

- Excel dosyasında hızlı arama
- Otomatik cache sistemi
- Kullanıcı dostu arayüz

## Kurulum

1. Telegram Bot Token alın (@BotFather)
2. Environment variables ayarlayın
3. Botu başlatın

## Environment Variables

- `TELEGRAM_BOT_TOKEN`: Telegram bot token
- `EXCEL_URL`: Excel dosyasının URL'i
